# Learning Roadmap

Phase 1: Behavior Explosion

- Growing if / switch statements
- Conditional logic tied to business behavior

Phase 2: Object Creation Pressure

- new scattered across the codebase
- Hard to change construction logic

Phase 3: Cross-cutting Concerns

- Logging, validation, decoration without modification

Phase 4: State Explosion

- Implicit state transitions
- Boolean flags controlling behavior

Phase 5: Reaction and Events

- Tight coupling between producer and consumer
